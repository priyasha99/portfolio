# Priyasha Setia — Portfolio

Live at [priyashasetia.com](https://www.priyashasetia.com)

## Concept

An interactive system architecture diagram — the entire page *is* the diagram. Each section of the portfolio is a draggable node on a whiteboard-style canvas, reflecting the backend engineering mindset behind the work it represents.

## Features

- Boot sequence on load with animated progress bar
- Draggable, rearrangeable nodes (desktop)
- Dark / light mode toggle with system preference detection
- SVG connection lines with animated data-flow dots between nodes
- Animated metric counters, live uptime clock and visitor count
- Floating particle canvas in the background
- Zoom in / out / reset controls
- Responsive layout — single column on mobile
- Subtle wiggle + cursor trail hint for drag discovery

## Tech Stack

- Vanilla HTML, CSS, JavaScript — zero build tools, zero dependencies
- Single `index.html` file
- Hosted on GitHub Pages with custom domain
